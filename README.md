# 📈 Stock AI Dashboard

An AI-powered stock analysis platform built with Streamlit.

This app combines machine learning, technical indicators, and backtesting to generate trading signals, analyze market data, and simulate portfolio performance in an interactive dashboard.

---

## 🚀 Live App

👉 https://stock-ai-pilot.streamlit.app

---

## ⚙️ Features

* 📊 **AI Trading Signals**
  Uses a Random Forest model to predict short-term price movement

* 📉 **Technical Indicators**
  RSI, MACD, SMA 50, SMA 200

* 🔍 **Stock Scanner**
  Scan multiple tickers and generate BUY / WAIT / DON'T BUY signals

* 🧪 **Backtesting Engine**
  Test strategies on historical data

* 💼 **Portfolio Simulator**
  Simulate trades and track performance

* 📰 **Market Data Integration**
  Live stock data using Yahoo Finance

---

## 🧠 How It Works

1. Fetch historical stock data
2. Generate technical indicators
3. Train machine learning model
4. Predict probability of price increase
5. Generate trading signals
6. Backtest and analyze results

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas / NumPy
* yFinance
* TA (Technical Analysis library)

## 📦 Installation (Local)

Clone the repo:

```bash
git clone https://github.com/R3AL1TY02/stock-ai.git
cd stock-ai
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run dashboard/app.py
```
